üöÄ Step-by-Step Installation

Step 1: Clone the Repository
git clone <repository-url>
cd digital-menu-card
Step 2: Install Dependencies
npm install
Step 3: Setup PostgreSQL Database
# Login to PostgreSQL
psql -U postgres

# Create database
CREATE DATABASE DigiMenu;

# Connect to database
\c DigiMenu

# Run schema file (if available)
\i routes/database.sql
Step 4: Configure Environment Variables
# Copy example file
cp .env.example .env

# Edit .env file with your credentials
nano .env

Required Environment Variables:

NODE_ENV=production
PORT=3000

# Database Configuration
DB_HOST=localhost
DB_USER=postgres
DB_PASSWORD=your_secure_password
DB_NAME=DigiMenu
DB_PORT=5432

# Message Central (OTP Service)
MESSAGE_CENTRAL_AUTH_KEY=your_auth_key
MESSAGE_CENTRAL_CUSTOMER_ID=your_customer_id

# Razorpay (Payment Gateway)
RAZORPAY_KEY_ID=your_key_id
RAZORPAY_KEY_SECRET=your_key_secret

# Bunny CDN (Image Hosting)
BUNNY_STORAGE_ZONE=your_storage_zone
BUNNY_ACCESS_KEY=your_access_key
BUNNY_CDN_URL=https://your-cdn-url.b-cdn.net
‚ö†Ô∏è Security Warning: Never commit the .env file to version control. It contains sensitive credentials. Always use .env.example as a template.
Step 5: Run the Application
# Development mode (with auto-reload)
npm run dev

# Production mode
npm start
‚úÖ Success! Server will start at http://localhost:3000
Customer Portal: http://localhost:3000
Admin Portal: http://localhost:3000/admin
